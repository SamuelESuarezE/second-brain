# 🧠 Second Brain

A personal knowledge management system (PKM) to document, organise and track everything I learn as a developer. The goal is not to copy documentation, it's to write things in my own words, build real examples, and have a honest record of my growth.

This repository also serves as public evidence of consistent, intentional learning.

---

## How I write notes

```markdown
# Topic name

## What is it
One or two lines in my own words.

## How it works
The minimum needed to understand it.

## My own example (when applicable)
```language
// code I wrote myself
​```

## When to use it
In which situations this applies.

## Resources 
- link to book chapter, article or video
```
